## 📌 Overview

Cairo Secure Eraser is a free Windows utility designed to permanently remove
files and sensitive data by using multi-pass overwriting techniques, reducing
the likelihood of file recovery compared to standard OS deletion methods.

---

## ✨ Features

- ✅ Permanently delete files and folders using multi-pass overwriting
- ✅ Multiple erasure methods with configurable overwrite passes
- ✅ Select specific files or folders for deletion
- ✅ Real-time progress tracking during deletion
- ✅ Batch processing — multiple files and folders in a single operation
- ✅ Detailed logs for all completed deletion operations
- ✅ Handles file and folder permissions before deletion
- ✅ Error handling mechanisms during deletion processes
- ✅ All operations run locally — no data leaves your device
- ✅ 100% Free for personal and commercial use

## 🛡️ Erasure Methods

| Method | Passes | Use Case |
|---|---|---|
| Quick | 1 pass | General cleanup |
| Standard | 3 passes | Personal files |
| Secure | 7 passes | Sensitive data |
| Military | 35 passes | Maximum security |

## 🚀 How to Use

1. **Download** the software and install it on your device
2. **Open** the software — the main interface will appear
3. **Add Files or Folders** using the Add Files or Add Folder button
4. **Choose Erasure Method** — Quick, Standard, Secure, or Military
5. **Click Erase Files** to start the permanent deletion process
6. **Monitor Progress** via the progress bar and status display
7. **Review Report** — a completion report will be shown after erasure
